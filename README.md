**Rasool Pro Hub** is a professional-grade cybersecurity dashboard designed for penetration testers, security researchers, and system administrators. It combines powerful network reconnaissance tools with file stealth technology and an intuitive project management interface.

## 🚀 Key Features

### 1. Advanced Vulnerability Scanner
- **Nmap Integration**: Deep service and version detection on target hosts.
- **AI-Powered Risk Reports**: Automatically generates security reports using Google Gemini AI to analyze scan results and provide actionable recommendations.
- **Real-time Geolocation**: Instantly identify the physical location and ISP of target IP addresses.

### 2. Stealth Vault (File Protection)
- **Windows Kernel API Integration**: Uses `SetFileAttributesW` to hide files at the system level (Hidden + System attributes).
- **Military-Grade Encryption**: Encrypts sensitive files using AES-128 (Fernet) before hiding them.
- **One-Click Restoration**: Easily unhide and decrypt files directly from the dashboard.

### 3. Professional Cyber Tools
- **Secure Key Generator**: Generate high-entropy encryption keys and passwords with customizable complexity.
- **Theme Switcher**: Choose between multiple premium themes (Midnight Dark, Emerald Stealth, Cyber Blue) to suit your working environment.
- **Smart Notification System**: Instant feedback on all security operations via an elegant toast notification system.

### 4. Integrated Project Manager
- Manage your coding projects, track status, and organize files in a centralized hub.

## 🛠 Tech Stack
- **Frontend**: React.js, Tailwind CSS, Lucide Icons, Axios.
- **Backend**: FastAPI (Python), Nmap, Cryptography (Fernet), Google Gemini AI API.
- **DevOps**: Docker & Docker Compose ready.

## 🏁 Quick Start

### Prerequisites
- Python 3.11+
- Node.js & NPM
- Nmap (must be installed on the host system)

### Automatic Setup (Windows)
1. Clone the repository to `C:\Users\HP\Desktop\Rasoolprohub`.
2. Run the `auto_setup.py` script to initialize the security modules.
3. Use the **`Run_My_Cyber_Site.bat`** on your desktop to launch both the backend and frontend with a single click.
